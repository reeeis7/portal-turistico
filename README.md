Trabalho -
Brasil Encantado:

Descrição - 
Projeto de desenvolvimento web que consiste na criação de um portal turístico voltado à divulgação de destinos brasileiros. O site foi desenvolvido utilizando HTML5 semântico e CSS externo, com foco em organização de conteúdo, navegação intuitiva e estrutura responsiva básica.

A proposta simula uma agência fictícia, “Brasil Encantado”, apresentando informações sobre cultura, gastronomia, pontos turísticos e experiências pelo país. O objetivo é oferecer uma experiência visual atrativa e facilitar o acesso às informações para turistas nacionais e internacionais.
